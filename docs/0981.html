铬黄弯弯曲曲地进入聚光灯下

原文：https://web.archive.org/web/https://dappradar.com/blog/chromie-squiggles-squiggle-into-the-spotlight

过去24小时内卖出了3幅铬黄画，每幅都超过了26万美元

在过去的24小时里，艺术街区项目的创世纪系列“三色曲线”登上了DappRadar网站NFT销售页面的首位。这三种非功能性食物的售价都超过了89 ETH，即26万美元以上。

令人印象深刻的是，在购买后，两条铬黄曲线被转移到同一个钱包。chrome Squiggle # 66和chrome Squiggle # 96前往开元宇宙系列由 punk6529 创作。本周早些时候，punk6529购买了Ringers #210 ，这是艺术街区策划系列的另一件作品。这两幅铬黄画的总价为639，758美元。

开放的元宇宙收藏馆现在收藏了222件价值超过1000万美元的NFT。用DappRadar Portfolio Tracker工具查看收藏的钱包地址，我们看到它包含了 36个艺术块NFTs 的各个子收藏部分。该投资组合中艺术板块的总价值达到172万美元，接近NFT钱包总价值的20%。

Chromie Squiggle #255 是过去24小时内该系列的第三次大拍卖。NFT以237，980美元的价格易手了89 ETH。NFT的新主人似乎刚刚开始这个特别的投资组合，到目前为止只有5件收藏品。即便如此，看起来铬黄曲线是这位收藏家的一个焦点，因为他们也拥有 #5620 。
